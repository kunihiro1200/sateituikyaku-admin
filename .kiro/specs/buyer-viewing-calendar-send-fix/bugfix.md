# バグ修正要件書

## Introduction

買主リストの内覧ページで内覧日のカレンダー送信をする時に、後続担当のカレンダーに送信せずに、送信者のアカウントに送信されてしまうバグを修正する。

## Bug Analysis

### Current Behavior (Defect)

1.1 WHEN 内覧ページで「📅 カレンダーで開く」ボタンを押す THEN システムは送信者のGoogleカレンダーを開き、送信者自身のカレンダーに予定を作成する

1.2 WHEN カレンダー送信が完了する THEN システムは後続担当（営業担当）のカレンダーに予定を送信しない

### Expected Behavior (Correct)

2.1 WHEN 内覧ページで「📅 カレンダーで開く」ボタンを押す THEN システムは送信者のGoogleカレンダーを開き、後続担当（営業担当）のカレンダーに予定を送信する

2.2 WHEN カレンダー送信が完了する THEN システムは「後続担当（〇〇）のGoogleカレンダーに内覧予約を登録しました」というメッセージを表示する

### Unchanged Behavior (Regression Prevention)

3.1 WHEN 売主リストの通話モードページで訪問日のカレンダー送信をする THEN システムは引き続き営担（visitAssignee）のカレンダーに正しく送信する

3.2 WHEN 内覧日・時間・後続担当が設定されている THEN システムは引き続きカレンダー送信ボタンを表示する

3.3 WHEN カレンダー送信後にGoogleカレンダーを開く THEN システムは引き続き送信者のGoogleカレンダーを開く
