# 物件URL手動編集機能 - クイックスタートガイド

## 🎉 実装完了！

物件詳細ページで地図URLと格納先URLを手動で編集できるようになりました。

## 📍 機能の場所

**物件詳細ページ** → **「地図・サイトURL」セクション**

このセクションは「配信エリア番号」セクションの直前にあります。

## 🚀 使い方

### 地図URLの編集

1. **物件詳細ページを開く**
   - 物件リストから任意の物件を選択

2. **「地図・サイトURL」セクションを見つける**
   - ページの中央部分にあります

3. **地図URLを編集**
   - 「地図URL」の横にある ✏️ 編集ボタンをクリック
   - Google Map URLを入力（例: `https://maps.google.com/maps?q=35.6812,139.7671`）
   - 「保存」ボタンをクリック

4. **自動処理**
   - 配信エリアが自動的に再計算されます
   - 成功メッセージが表示されます

### 格納先URLの編集

1. **物件詳細ページを開く**
   - 物件リストから任意の物件を選択

2. **「地図・サイトURL」セクションを見つける**
   - ページの中央部分にあります

3. **格納先URLを編集**
   - 「格納先URL」の横にある ✏️ 編集ボタンをクリック
   - Google DriveフォルダURLを入力（例: `https://drive.google.com/drive/folders/1a2b3c4d5e6f`）
   - 「保存」ボタンをクリック

4. **完了**
   - 成功メッセージが表示されます

## ✅ 有効なURL形式

### Google Map URL
以下のいずれかの形式が有効です：
- `https://maps.google.com/...`
- `https://www.google.com/maps/...`
- `https://goo.gl/maps/...`（短縮URL）

### Google DriveフォルダURL
以下の形式が有効です：
- `https://drive.google.com/drive/folders/...`
- `https://drive.google.com/drive/u/0/folders/...`

## 💡 ヒント

- **空のURLも有効**: URLを削除したい場合は、空欄にして保存できます
- **リアルタイム検証**: 無効なURL形式を入力すると、すぐにエラーメッセージが表示されます
- **配信エリア自動更新**: 地図URLを変更すると、配信エリアが自動的に再計算されます
- **クリック可能なリンク**: 保存されたURLは、クリックして新しいタブで開くことができます

## ⚠️ 注意事項

1. **URL形式を確認**: 上記の有効な形式に従ってください
2. **保存を忘れずに**: 編集後は必ず「保存」ボタンをクリックしてください
3. **キャンセル可能**: 間違えた場合は「キャンセル」ボタンで元に戻せます

## 🔧 トラブルシューティング

### エラー: 「有効なGoogle Map URLを入力してください」
- URL形式が正しいか確認してください
- `https://`で始まっているか確認してください
- Google Mapsのドメインが正しいか確認してください

### エラー: 「有効なGoogle DriveフォルダURLを入力してください」
- URL形式が正しいか確認してください
- `https://drive.google.com/drive/folders/`で始まっているか確認してください
- フォルダIDが含まれているか確認してください

### 保存できない
- ネットワーク接続を確認してください
- ブラウザのコンソールでエラーメッセージを確認してください
- ページをリロードして再試行してください
