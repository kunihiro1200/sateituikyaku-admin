# Athome情報セクション - ユーザーガイド

## 概要

物件公開サイトの詳細ページに表示される「物件詳細情報」セクションについての説明です。

## 表示場所

物件詳細ページの「物件基本情報」セクションの直後に表示されます。

```
┌─────────────────────────────┐
│ 物件画像ギャラリー           │
└─────────────────────────────┘
┌─────────────────────────────┐
│ 物件基本情報                 │
│ - 価格                       │
│ - 住所                       │
│ - 土地面積                   │
│ - 建物面積                   │
│ - 築年数                     │
└─────────────────────────────┘
┌─────────────────────────────┐
│ 物件詳細情報 ← ここ！        │
│ ● 駅から徒歩5分              │
│ ● 南向きで日当たり良好       │
│ 閑静な住宅街                 │
└─────────────────────────────┘
```

## 表示内容

物件ごとに異なる詳細情報が表示されます:

- **重要なポイント**: ●マークで始まる項目
- **その他の情報**: マークなしの項目

### 表示例

```
物件詳細情報
● 駅から徒歩5分の好立地
● 南向きで日当たり良好
● リフォーム済み
閑静な住宅街
スーパー・コンビニ徒歩圏内
```

## 表示されない場合

以下の場合、このセクションは表示されません:

1. **物件に詳細情報が登録されていない場合**
   - データがない物件では非表示になります

2. **一時的なエラーが発生した場合**
   - ネットワークエラーなどで取得できない場合は非表示になります
   - ページの他の部分は正常に表示されます

## よくある質問

### Q: 詳細情報を更新したい

A: 詳細情報は業務リストスプレッドシートの「athome」シートで管理されています。更新が必要な場合は、担当者にご連絡ください。

### Q: ●マークの意味は？

A: ●マークは特に重要なポイントを示しています。物件の主要な特徴や魅力を表しています。

### Q: 情報が古い

A: 情報は最大5分間キャッシュされます。最新の情報を確認したい場合は、5分後にページを再読み込みしてください。

### Q: スマートフォンでも見られる？

A: はい、スマートフォンやタブレットでも最適化された表示で閲覧できます。

## 管理者向け情報

### データの更新方法

1. 業務リストスプレッドシートを開く
2. 「athome」シート内の該当物件のシートを開く
3. 物件タイプに応じたセル範囲を編集:
   - **土地**: B63〜B79
   - **戸建て**: B152〜B166
   - **マンション**: B149〜B163
4. 保存後、最大5分でサイトに反映されます

### 記号の使い方

- **★（星マーク）**: 入力すると自動的に●（黒丸）に変換されます
- **空行**: 空のセルは表示されません
- **改行**: セルごとに1行として表示されます

### 注意事項

- セル範囲外のデータは表示されません
- 物件番号と同じ名前のシートが必要です
- スプレッドシートへのアクセス権限が必要です
