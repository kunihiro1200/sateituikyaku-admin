# バグ修正要件ドキュメント

## はじめに

買主リスト詳細画面において、他社物件フィールドに値がない場合でも「他社物件情報」セクションが表示されている問題を修正します。新規買主登録画面では、他社物件に値がなくても常に「他社物件情報」セクションを表示する必要があります（入力のため）。

## バグ分析

### 現在の動作（不具合）

1.1 WHEN 買主詳細画面で他社物件フィールドに値がない THEN システムは「他社物件情報」セクションを表示する

### 期待される動作（正しい動作）

2.1 WHEN 買主詳細画面で他社物件フィールドに値がある THEN システムは「他社物件情報」セクションを表示する

2.2 WHEN 買主詳細画面で他社物件フィールドに値がない THEN システムは「他社物件情報」セクションを非表示にする

2.3 WHEN 新規買主登録画面を表示する THEN システムは他社物件フィールドの値の有無に関わらず「他社物件情報」セクションを表示する

### 変更されない動作（リグレッション防止）

3.1 WHEN 買主詳細画面で他社物件フィールドに値を入力する THEN システムは引き続き正常に保存する

3.2 WHEN 新規買主登録画面で他社物件フィールドに値を入力する THEN システムは引き続き正常に保存する

3.3 WHEN 買主詳細画面で他の全てのセクション（基本情報、問合せ内容など）を表示する THEN システムは引き続き正常に表示する
