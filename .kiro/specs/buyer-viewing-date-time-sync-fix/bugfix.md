# バグ条件: 買主リストの内覧日・時間が同期されない

## バグの症状

買主リストにおいて、スプレッドシートの「●内覧日(最新)」（I列）と「●時間」（BP列）がデータベースに同期されない。

**影響範囲**: 全買主（7282を含む全体的な問題）

**発見日**: 2026年4月3日

---

## バグ条件 C(X)

**入力 X**: 買主リストスプレッドシートの行データ

**バグ条件 C(X)**:
```
GASの syncBuyerList() が実行される
AND スプレッドシートに「●時間」カラムが存在する
AND GASコードが row['時間'] でカラムを参照している
→ カラム名の不一致により、viewing_time が同期されない
```

**根本原因**:
- GASコード（496行目）が `row['時間']` でカラムを参照
- 実際のスプレッドシートのカラム名は `●時間`（BP列）
- カラム名が一致しないため、`row['時間']` は常に `undefined` を返す
- 結果として `viewing_time` が同期されない

---

## 期待される動作

**正しい動作**:
1. GASが `row['●時間']` でスプレッドシートのBP列を参照
2. 値が取得できる
3. `viewing_time` がデータベースに正しく同期される

---

## 実際の動作

**現在の動作**:
1. GASが `row['時間']` でカラムを参照
2. カラム名が一致しないため `undefined` が返される
3. `viewing_time` が同期されない（または null になる）

---

## 再現手順

1. 買主リストスプレッドシートで買主7282の行を確認
2. I列「●内覧日(最新)」に `2026/04/05` が入力されている
3. BP列「●時間」に `14:30` が入力されている
4. GASの `syncBuyerList()` を実行
5. データベースを確認
6. `viewing_date` は null、`viewing_time` は同期されていない

---

## 影響を受けるデータ

- **テーブル**: `buyers`
- **カラム**: `viewing_time`
- **影響範囲**: 全買主（いつから発生したかは不明）

## 修正方針

### 修正1: GASコードのカラム名を修正

**ファイル**: `gas_buyer_complete_code.js`

**修正箇所**: 496行目

**変更前**:
```javascript
var sheetViewingTime = row['時間'] ? String(row['時間']) : null;
```

**変更後**:
```javascript
var sheetViewingTime = row['●時間'] ? String(row['●時間']) : null;
```
