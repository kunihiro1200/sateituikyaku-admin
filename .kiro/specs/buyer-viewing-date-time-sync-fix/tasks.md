## タスク詳細

### タスク1: GASコードを修正

#### 1.1 `gas_buyer_complete_code.js` の496行目を修正

**変更前**:
```javascript
var sheetViewingTime = row['時間'] ? String(row['時間']) : null;
```

**変更後**:
```javascript
var sheetViewingTime = row['●時間'] ? String(row['●時間']) : null;
```

**手順**:
1. `gas_buyer_complete_code.js` を開く
2. 496行目を探す
3. `row['時間']` を `row['●時間']` に変更（2箇所）
4. コメントを更新: `// 時間` → `// 時間（BP列「●時間」）`
5. 保存

#### 1.2 Google Apps Scriptエディタにコピー＆ペースト

**手順**:
1. 買主リストスプレッドシートを開く
2. 「拡張機能」→「Apps Script」を選択
3. `gas_buyer_complete_code.js` の内容を**全て**コピー
4. GASエディタに**全て**ペースト（既存コードを上書き）
5. 保存（Ctrl+S）
