# バグ修正要件書

## Introduction

売主リストの検索機能において、売主番号のプレフィックスが`AA`のみに限定されており、`FI`、`BB`、`CC`等の他のアルファベット2文字プレフィックスを持つ売主番号が検索できない問題を修正します。

現在の実装では正規表現`/^aa\d+$/i`を使用しているため、`AA`プレフィックスのみが検索可能です。これを`/^[a-z]{2}\d+$/i`に変更することで、任意の2文字アルファベットプレフィックスに対応します。

## Bug Analysis

### Current Behavior (Defect)

1.1 WHEN 売主番号検索バーに`FI123`を入力 THEN システムは検索結果を返さない

1.2 WHEN 売主番号検索バーに`BB456`を入力 THEN システムは検索結果を返さない

1.3 WHEN 売主番号検索バーに`CC789`を入力 THEN システムは検索結果を返さない

1.4 WHEN 売主番号検索バーに`AA`以外の2文字アルファベットプレフィックス + 数字の組み合わせを入力 THEN システムは正規表現マッチに失敗し、検索が実行されない

### Expected Behavior (Correct)

2.1 WHEN 売主番号検索バーに`FI123`を入力 THEN システムSHALL正規表現にマッチし、`FI123`の売主を検索結果として返す

2.2 WHEN 売主番号検索バーに`BB456`を入力 THEN システムSHALL正規表現にマッチし、`BB456`の売主を検索結果として返す

2.3 WHEN 売主番号検索バーに`CC789`を入力 THEN システムSHALL正規表現にマッチし、`CC789`の売主を検索結果として返す

2.4 WHEN 売主番号検索バーに任意の2文字アルファベットプレフィックス + 数字の組み合わせを入力 THEN システムSHALL正規表現`/^[a-z]{2}\d+$/i`にマッチし、該当する売主を検索結果として返す

### Unchanged Behavior (Regression Prevention)

3.1 WHEN 売主番号検索バーに`AA13501`を入力 THEN システムSHALL CONTINUE TO正規表現にマッチし、`AA13501`の売主を検索結果として返す

3.2 WHEN 売主番号検索バーに`AA`プレフィックスを持つ売主番号を入力 THEN システムSHALL CONTINUE TO正常に検索を実行し、結果を返す

3.3 WHEN 売主番号検索バーに無効な形式（例: `A123`、`ABC123`、`12345`）を入力 THEN システムSHALL CONTINUE TO正規表現にマッチせず、検索を実行しない

3.4 WHEN 売主番号以外の検索条件（名前、住所等）で検索 THEN システムSHALL CONTINUE TO正常に検索を実行し、結果を返す
