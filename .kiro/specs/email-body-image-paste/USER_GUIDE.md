# メール本文への画像貼り付け機能 - ユーザーガイド

## 概要

メール送信確認画面で、スクリーンショットなどの画像を直接本文に貼り付けることができます。

## 使い方

### 基本的な使い方

1. **コールモードページ**でメール送信確認画面を開く
2. メール本文のテキストフィールドにフォーカス
3. スクリーンショットをコピー（PrintScreenキーなど）
4. **Ctrl+V**（Windows）または**Cmd+V**（Mac）で貼り付け
5. プレビューで画像を確認
6. メール送信ボタンをクリック

### 画像の配置方法

#### 方法1: 本文の最後に自動配置

画像を貼り付けると、自動的に本文の最後に追加されます。

```
お世話になっております。

物件の写真をお送りします。

[ここに画像が自動的に表示されます]
```

#### 方法2: 任意の場所に配置（プレースホルダー使用）

本文内に `[画像1]`、`[画像2]` などのプレースホルダーを入力すると、その位置に画像が表示されます。

**例：**

```
お世話になっております。

外観写真です：
[画像1]

室内写真です：
[画像2]

よろしくお願いいたします。
```

画像を貼り付けた順番で、`[画像1]`、`[画像2]`... と対応します。

### 画像の削除

プレビューエリアの各画像カードにある「削除」ボタンをクリックすると、その画像を削除できます。

## 制限事項

- **1枚あたりの最大サイズ**: 5MB
- **合計最大サイズ**: 10MB
- **対応形式**: PNG、JPEG、GIF、WebP など（ブラウザが対応している画像形式）

## トラブルシューティング

### 画像が貼り付けられない

- クリップボードに画像がコピーされているか確認してください
- ファイルサイズが5MBを超えていないか確認してください
- ブラウザのコンソールにエラーが表示されていないか確認してください

### 件名が文字化けする

- 最新版では修正されています
- バックエンドサーバーを再起動してください

### 画像の位置がおかしい

- プレースホルダー（`[画像1]`など）を使用して、明示的に位置を指定してください
- プレースホルダーがない場合は、自動的に本文の最後に追加されます

## 技術的な詳細

- 画像はBase64エンコードされてHTMLメールに埋め込まれます
- メールクライアント（Gmail、Outlookなど）で正しく表示されます
- 画像データはメール本体に含まれるため、別途添付ファイルとして送信されません
