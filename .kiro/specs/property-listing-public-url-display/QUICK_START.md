# 物件リスト公開URL表示機能 - クイックスタートガイド

## 🚀 3ステップで顧客にURLを共有

### ステップ1: 物件リストを開く
サイドメニューから「物件リスト」を選択

### ステップ2: URLをコピー
公開中物件の「公開URL」カラムにあるコピーボタン（📋）をクリック

### ステップ3: 顧客に送信
メール、LINE、SMSなどにURLを貼り付けて送信

---

## 📋 機能概要

| 機能 | 説明 |
|------|------|
| **URL表示** | 公開中物件のURLを短縮形式で表示 |
| **ツールチップ** | URLにホバーで完全URLを表示 |
| **ワンクリックコピー** | コピーボタンで即座にクリップボードにコピー |
| **視覚的フィードバック** | コピー成功時にチェックマーク表示 |
| **トースト通知** | 成功/エラーを画面下部に表示 |

---

## 🎯 使用例

### 例1: 新規顧客にURLを送る
```
1. 物件リストで「専任・公開中」の物件を探す
2. コピーボタンをクリック
3. メールに貼り付けて送信
```

### 例2: 複数物件のURLを一括送信
```
1. 物件A のURLをコピー → メモ帳に貼り付け
2. 物件B のURLをコピー → メモ帳に貼り付け
3. 物件C のURLをコピー → メモ帳に貼り付け
4. メモ帳の内容をメールに貼り付けて送信
```

### 例3: 担当物件のみを表示
```
1. 左サイドバーの「担当者」から自分の名前を選択
2. 「ステータス」から「公開中」を選択
3. 自分が担当する公開中物件のみが表示される
```

## ✅ チェックリスト

### URLが表示されない場合
- [ ] 物件の「ATBB状況」が「専任・公開中」になっているか確認
- [ ] ページを再読み込みしてみる
- [ ] ブラウザのキャッシュをクリアしてみる

### コピーできない場合
- [ ] ブラウザが最新版か確認
- [ ] ブラウザのクリップボード権限を確認
- [ ] 別のブラウザで試してみる

---

## 💡 ヒント

### 効率的な使い方
- **フィルター活用**: 「公開中」でフィルターして公開物件のみ表示
- **検索機能**: 物件番号で検索して素早く見つける
- **ホバー確認**: URLにマウスを乗せて完全URLを確認

### 顧客への共有
- **メール**: 件名に物件番号を入れると分かりやすい
- **LINE**: URLの前に簡単な説明を添える
- **SMS**: 短縮URLなので文字数制限に収まりやすい

---

## 🔧 トラブルシューティング

| 問題 | 解決策 |
|------|--------|
| URLが「-」と表示 | ATBB状況を「専任・公開中」に変更 |
| コピーボタンが反応しない | ページを再読み込み |
| 「コピーに失敗しました」 | ブラウザのクリップボード権限を許可 |
