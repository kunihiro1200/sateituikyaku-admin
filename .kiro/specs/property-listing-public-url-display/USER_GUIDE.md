# 物件リスト公開URL表示機能 - ユーザーガイド

## 概要

物件リスト画面に「公開URL」カラムが追加され、公開中の物件のURLを簡単にコピーして顧客に共有できるようになりました。

## 機能説明

### 1. 公開URL表示

物件リスト画面の「公開URL」カラムに、公開中物件のURLが表示されます。

- **公開中物件**: 短縮URLが表示されます（例: `...abc-123-def-456`）
- **非公開物件**: 「-」が表示されます

### 2. URL確認

URLにマウスをホバーすると、完全なURLがツールチップで表示されます。

例: `https://your-domain.com/public/properties/abc-123-def-456`

### 3. URLコピー

コピーボタン（📋アイコン）をクリックすると、URLがクリップボードにコピーされます。

#### コピー成功時
- アイコンがチェックマーク（✓）に変わります
- 画面下部に「URLをコピーしました」と表示されます
- 3秒後に元のアイコンに戻ります

#### コピー失敗時
- 画面下部に「コピーに失敗しました」と表示されます

## 使い方

### 基本的な使い方

1. **物件リスト画面を開く**
   - サイドメニューから「物件リスト」を選択

2. **公開中物件を確認**
   - 「ATBB状況」カラムが「専任・公開中」の物件を探す
   - 「公開URL」カラムにURLが表示されている

3. **URLをコピー**
   - コピーボタン（📋）をクリック
   - 「URLをコピーしました」と表示されることを確認

4. **顧客に共有**
   - メール、LINE、SMSなどにURLを貼り付けて送信

### フィルター機能との組み合わせ

#### ステータスフィルター
左サイドバーの「ステータス」から「公開中」を選択すると、公開中物件のみが表示されます。

#### 担当者フィルター
左サイドバーの「担当者」から自分の名前を選択すると、自分が担当する物件のみが表示されます。

#### 検索機能
画面上部の検索ボックスで物件番号や所在地を検索できます。

### 複数物件のURL共有

1. 複数の物件のURLをコピーしたい場合は、1件ずつコピーボタンをクリックします
2. 各URLをメモ帳などに貼り付けて整理します
3. 顧客に一括で送信します

## よくある質問（FAQ）

### Q1: URLが表示されない物件があります
**A:** 「ATBB状況」が「専任・公開中」以外の物件は、URLが表示されません（「-」と表示されます）。物件を公開するには、ATBB状況を「専任・公開中」に変更してください。

### Q2: コピーボタンが反応しません
**A:** 以下を確認してください：
- ブラウザが最新版か確認
- ブラウザの設定でクリップボードへのアクセスが許可されているか確認
- ページを再読み込みして再度試す

### Q3: コピーしたURLが正しく貼り付けられません
**A:** コピー後すぐに貼り付けてください。他のテキストをコピーすると、クリップボードの内容が上書きされます。

### Q4: URLをクリックしても物件詳細が開きません
**A:** URLは顧客向けの公開サイトのURLです。管理画面で物件詳細を見るには、物件番号をクリックしてください。

### Q5: URLの短縮表示が見づらいです
**A:** URLにマウスをホバーすると、完全なURLがツールチップで表示されます。

### Q6: 古いブラウザでも使えますか？
**A:** はい。最新のClipboard APIに対応していないブラウザでも、フォールバック機能により動作します。

## トラブルシューティング

### 問題: コピーボタンをクリックしても何も起こらない

**原因1**: ブラウザのクリップボード権限が拒否されている
**解決策**: ブラウザの設定でクリップボードへのアクセスを許可してください

**原因2**: JavaScriptが無効になっている
**解決策**: ブラウザの設定でJavaScriptを有効にしてください

### 問題: 「コピーに失敗しました」と表示される

**原因**: ブラウザのセキュリティ設定により、クリップボードへのアクセスがブロックされている
**解決策**: 
1. ブラウザのアドレスバーの鍵アイコンをクリック
2. 「サイトの設定」を開く
3. 「クリップボード」を「許可」に変更

### 問題: URLが「-」と表示される

**原因**: 物件が公開されていない
**解決策**: 
1. 物件詳細画面を開く
2. 「ATBB状況」を「専任・公開中」に変更
3. 保存して物件リストに戻る

## セキュリティとプライバシー

### URLの公開範囲
- 公開URLは誰でもアクセス可能です
- URLを知っている人は物件情報を閲覧できます
- 顧客以外には共有しないでください

### URLの有効期限
- URLに有効期限はありません
- 物件が非公開になると、URLにアクセスしても物件情報は表示されません

## ヒントとコツ

### 効率的な使い方
1. **フィルターを活用**: 「公開中」ステータスでフィルターして、公開中物件のみを表示
2. **検索機能**: 物件番号で検索して、特定の物件を素早く見つける
3. **ソート機能**: 契約日でソートして、最新の公開物件を確認

### 顧客への共有方法
1. **メール**: URLをメール本文に貼り付けて送信
2. **LINE**: URLをLINEメッセージに貼り付けて送信
3. **SMS**: URLをSMSに貼り付けて送信
4. **QRコード**: URLをQRコード生成サイトでQRコードに変換して印刷
