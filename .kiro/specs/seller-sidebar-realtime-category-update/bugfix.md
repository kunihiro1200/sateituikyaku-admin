# バグ修正要件書

## Introduction

売主リストのサイドバーカテゴリーにおいて、通話モードページで次電日を変更した後、当日TEL一覧に戻ると、条件から外れた案件が約5秒間残り続けるバグを修正します。この遅延により、ユーザーが次の案件に電話しようとする際に混乱が生じています。

## Bug Analysis

### Current Behavior (Defect)

1.1 WHEN 通話モードページで次電日を変更して当日TEL一覧に戻る THEN 変更した案件が約5秒間サイドバーに残り続ける

1.2 WHEN 次電日を今日以降の日付に変更する THEN 当日TEL分カテゴリーから即座に除外されず、約5秒後に自動的に消える

1.3 WHEN サイドバーカテゴリーの条件から外れる変更を行う THEN 全てのサイドバーカテゴリーで同様の遅延が発生する可能性がある

### Expected Behavior (Correct)

2.1 WHEN 通話モードページで次電日を変更して当日TEL一覧に戻る THEN 変更した案件は即座にサイドバーから除外されるべき

2.2 WHEN 次電日を今日以降の日付に変更する THEN 当日TEL分カテゴリーから即座に除外され、遅延なく正しいカテゴリーに表示されるべき

2.3 WHEN サイドバーカテゴリーの条件から外れる変更を行う THEN 全てのサイドバーカテゴリーで即座に反映されるべき

### Unchanged Behavior (Regression Prevention)

3.1 WHEN 次電日を変更せずに当日TEL一覧に戻る THEN サイドバーの表示は変更されないまま維持されるべき

3.2 WHEN 次電日を今日以前の日付に変更する THEN 当日TEL分カテゴリーに正しく表示され続けるべき

3.3 WHEN サイドバーカテゴリーの条件を満たす変更を行う THEN 該当するカテゴリーに正しく表示され続けるべき

3.4 WHEN ページをリロードする THEN サイドバーのカテゴリー判定は正しく動作し続けるべき
