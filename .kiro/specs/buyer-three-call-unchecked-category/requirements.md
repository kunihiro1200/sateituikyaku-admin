# 買主リスト「３回架電未」カテゴリ追加 - 要件定義

## 📋 概要

買主リストのサイドバーに新しいカテゴリ「３回架電未」を追加します。このカテゴリは「当日TEL」カテゴリの下に表示され、３回架電確認が未完了の買主を表示します。

---

## 🎯 ビジネス要件

### 目的
- ３回架電確認が未完了の買主を可視化する
- 「当日TEL」カテゴリの下に配置することで、優先度の高い架電対象を明確にする

### 対象ユーザー
- 営業担当者
- 管理者

---

## 📊 機能要件

### 1. サイドバーカテゴリの追加

**カテゴリ名**: `３回架電未`

**表示位置**: 「当日TEL」カテゴリの直下

**表示順序**:
```
All: 4683
②内覧日前日: 2
⑯当日TEL: 2
  ↳ ３回架電未: 5  ← 新規追加
担当(Y): 150
  ↳ 当日TEL(Y): 10
```

**色**: 赤（#d32f2f）

**インデント**: あり（「当日TEL」の下に表示）

---

### 2. 表示条件

#### 2-1. 「当日TEL」カテゴリの条件

**条件**（スプレッドシートの数式から）:
```
IFS(
  AND(
    ISBLANK([後続担当]),
    [★次電日] <= TODAY(),
    ISNOTBLANK([★次電日])
  ),
  "⑯当日TEL"
)
```

**詳細**:
1. **後続担当**が**空**（ISBLANK）
2. **かつ** **★次電日** <= **今日**
3. **かつ** **★次電日**が**空でない**（ISNOTBLANK）

#### 2-2. 「３回架電未」カテゴリの条件

**条件**（スプレッドシートの数式から）:
```
AND(
  [3回架電確認済み] = "3回架電未",
  OR(
    [【問合メール】電話対応] = "不通",
    [【問合メール】電話対応] = "未"
  )
)
```

**詳細**:
1. **FC列「３回架電確認済み」** = `"3回架電未"`
2. **かつ** 以下のいずれか：
   - **「【問合メール】電話対応」** = `"不通"`
   - **「【問合メール】電話対応」** = `"未"`

**重要**:
- 「３回架電未」は「当日TEL」のサブセットではない（独立したカテゴリ）
- 両方の条件を満たす必要がある（AND条件）

---

### 3. データソース

**スプレッドシート**:

| 列 | カラム名 | データベースカラム名 | 説明 |
|----|---------|-------------------|------|
| **列位置未確認** | `後続担当` | `follow_up_assignee` | 後続担当者のイニシャル |
| **列位置未確認** | `★次電日` | `next_call_date` | 次回電話日 |
| **FC列** | `３回架電確認済み` | `three_call_confirmed` | 3回架電の確認状態（"3回架電未" or その他） |
| **列位置未確認** | `【問合メール】電話対応` | `inquiry_email_phone_response` | 電話対応状態（"不通", "未", その他） |

**データベース**:
- テーブル: `buyers`
- 既存カラム1: `follow_up_assignee` TEXT（後続担当）
- 既存カラム2: `next_call_date` DATE（次電日）
- 新規カラム1: `three_call_confirmed` TEXT（"3回架電未" or その他）
- 新規カラム2: `inquiry_email_phone_response` TEXT（"不通", "未", その他）

**注意**: 
- `follow_up_assignee`と`next_call_date`は既存カラム（既にデータベースに存在）
- 「【問合メール】電話対応」カラムの列位置は要確認

---

### 4. カウント表示

**表示形式**: `↳ ３回架電未: 5`

**カウント対象**:
- 「３回架電確認済み」が空
- かつ「当日TEL」の条件を満たす買主

### 互換性

- 既存のGAS同期処理に影響を与えない

## 🚨 制約事項

### 技術的制約
- FC列（「３回架電確認済み」）は買主リストスプレッドシートに既に存在する
- 「【問合メール】電話対応」カラムの列位置は要確認
- データベースに2つの新しいカラムを追加する必要がある：
  - `three_call_confirmed`
  - `inquiry_email_phone_response`

### ビジネス的制約
- 「３回架電確認済み」フィールドの入力は手動（スプレッドシート上で入力）
- 「【問合メール】電話対応」フィールドの入力は手動（スプレッドシート上で入力）
- 自動的に「確認済み」にする機能は含まない

## 📝 補足情報

### 表示条件の詳細

「３回架電未」カテゴリは以下の2つの条件を**両方**満たす買主を表示します：

1. **３回架電確認済み** = `"3回架電未"`
2. **【問合メール】電話対応** = `"不通"` **または** `"未"`

**例**:

| 買主番号 | ３回架電確認済み | 【問合メール】電話対応 | 表示 |
|---------|----------------|---------------------|------|
| 7001 | `"3回架電未"` | `"不通"` | ✅ 表示 |
| 7002 | `"3回架電未"` | `"未"` | ✅ 表示 |
| 7003 | `"3回架電未"` | `"対応済み"` | ❌ 非表示 |
| 7004 | `"確認済み"` | `"不通"` | ❌ 非表示 |
| 7005 | `""` (空) | `"不通"` | ❌ 非表示 |

### スプレッドシート列位置

| 列名 | カラム名 | 列位置（要確認） |
|------|---------|----------------|
| **FC** | `３回架電確認済み` | FC列 |
| **未確認** | `【問合メール】電話対応` | 要確認 |

**注意**: 「【問合メール】電話対応」カラムの正確な列位置は、スプレッドシートを確認して特定する必要があります。
