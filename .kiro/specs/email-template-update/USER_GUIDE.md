# Email Template Feature - User Guide

## 概要

通話モードページのEmail送信機能が更新され、21種類の新しいテンプレートが利用可能になりました。

## 主な機能

### 1. テンプレート選択

通話モードページの「Email送信」ドロップダウンから、適切なテンプレートを選択できます。

**表示される情報:**
- **種別**: テンプレートの名前（例：訪問査定後御礼メール）
- **件名**: メールの件名
- **本文プレビュー**: メール本文の最初の2行

### 2. サイト別フィルタリング

売主のサイト情報に基づいて、関連するテンプレートのみが表示されます。

**サイト別テンプレート:**
- **ウ（イエウール）**: イエウール専用テンプレート + 共通テンプレート
- **L/Y（LIFULL/Yahoo）**: LIFULL/Yahoo専用テンプレート + 共通テンプレート
- **す（すまいステップ）**: すまいステップ専用テンプレート + 共通テンプレート
- **H（HOME4U）**: HOME4U専用テンプレート + 共通テンプレート
- **その他**: 共通テンプレートのみ

### 3. 動的な表示順序

状況に応じて、重要なテンプレートが自動的に上位に表示されます。

**優先表示される条件:**
1. **訪問日が1週間以内** → 「訪問査定後御礼メール」が最上位に表示
2. **ステータスに「他決」が含まれる** → 「他決→追客」テンプレートが最上位に表示
3. **訪問予定日の1-2日前** → 「☆訪問前日通知メール」が最上位に表示

### 4. 自動プレースホルダー置換

テンプレート内のプレースホルダーは、売主・物件・担当者情報で自動的に置き換えられます。

**基本プレースホルダー:**
- `<<名前(漢字のみ）>>` → 売主氏名
- `<<物件所在地>>` → 物件住所
- `<<査定額1>>`, `<<査定額2>>`, `<<査定額3>>` → 査定額（万円）
- `<<土（㎡）>>`, `<<建（㎡）>>` → 土地・建物面積
- `<<営担>>` → 担当営業名
- `<<担当名（営業）電話番号>>` → 担当者電話番号
- `<<担当名（営業）メールアドレス>>` → 担当者メールアドレス
- `<<訪問日>>`, `<<時間>>` → 訪問予定日時
- `<<競合名>>` → 競合会社名

**条件付きプレースホルダー:**
- `<<築年不明>>` → 戸建てで築年不明の場合のみメッセージを表示
- `<<お客様紹介文言>>` → マンションと戸建て/土地で異なる文言を表示

### 5. 送信前確認

テンプレートを選択すると、確認ダイアログが表示されます。

**確認内容:**
- 送信先メールアドレス
- 件名（プレースホルダー置換済み）
- 本文（プレースホルダー置換済み）
- **URLはクリック可能なリンクとして表示されます**

## 利用手順

1. **通話モードページを開く**
   - 売主詳細ページから「通話モード」ボタンをクリック

2. **テンプレートを選択**
   - 「Email送信」ドロップダウンから適切なテンプレートを選択
   - サイト情報や状況に応じて、関連テンプレートが表示されます

3. **内容を確認**
   - 確認ダイアログで件名と本文を確認
   - プレースホルダーが正しく置き換えられているか確認
   - URLをクリックして内容を確認可能

4. **送信**
   - 「送信」ボタンをクリックしてメールを送信
   - 活動履歴に自動的に記録されます

5. **キャンセル**
   - 内容に問題がある場合は「キャンセル」ボタンをクリック

## 注意事項

1. **メールアドレスが必須**
   - 売主のメールアドレスが登録されていない場合、Email送信ドロップダウンは無効化されます

2. **査定額が必要なテンプレート**
   - 査定額を含むテンプレートを使用する場合は、事前に査定計算を実行してください

3. **担当者情報**
   - 担当者が割り当てられていない場合は、現在ログイン中のユーザー情報が使用されます

4. **訪問日時情報**
   - 訪問予定日時が設定されていない場合、該当プレースホルダーは空白になります

## トラブルシューティング

### テンプレートが表示されない
- 売主のサイト情報を確認してください
- サイト専用テンプレートは該当サイトでのみ表示されます

### プレースホルダーが置き換えられない
- 売主・物件情報が正しく登録されているか確認してください
- 欠損データがある場合、該当箇所は空白または「-」で表示されます

### メールが送信できない
- 売主のメールアドレスが正しく登録されているか確認してください
- ネットワーク接続を確認してください
