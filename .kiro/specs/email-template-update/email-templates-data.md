## 実装方法

これらのテンプレートは、CallModePage.tsxの`emailTemplates`配列に実装されます。

各テンプレートは以下の形式で定義されます：

```typescript
{
  id: string,        // ユニークID（例: 'visit_thank_you'）
  label: string,     // 種別（ドロップダウンに表示）
  subject: string,   // 件名（プレースホルダーを含む）
  content: string    // 本文（プレースホルダーを含む）
}
```

## プレースホルダー

以下のプレースホルダーがテンプレート内で使用されます：

- `<<名前(漢字のみ）>>` - 売主氏名
- `<<物件所在地>>` - 物件住所
- `<<査定額1>>`, `<<査定額2>>`, `<<査定額3>>` - 査定額（万円）
- `<<土（㎡）>>`, `<<建（㎡）>>` - 土地・建物面積
- `<<築年不明>>` - 築年情報
- `<<営担>>`, `<<担当名（営業）名前>>` - 担当者名
- `<<担当名（営業）電話番号>>` - 担当者電話番号
- `<<担当名（営業）メールアドレス>>` - 担当者メールアドレス
- `<<訪問日>>`, `<<時間>>` - 訪問日時
- `<<競合名>>` - 競合会社名
- `<<お客様紹介文言>>` - カスタムテキスト
