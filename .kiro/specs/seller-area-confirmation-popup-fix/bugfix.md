# Bugfix Requirements Document

## Introduction

売主番号AA13888の通話モードページにおいて、査定計算セクションで固定資産税路線価を入力しようとすると、「土地面積が30㎡（約9坪）ですが確認大丈夫ですか？」というポップアップが繰り返し表示され、固定資産税路線価の入力ができない問題を修正する。

ユーザーが「確認しました」ボタンを押した後も、固定資産税路線価フィールドをクリックするたびに同じポップアップが再表示されるため、実質的に入力が不可能な状態になっている。

## Bug Analysis

### Current Behavior (Defect)

1.1 WHEN 固定資産税路線価フィールドに値を入力する THEN システムは土地面積が99㎡以下または建物面積より小さい場合に警告ポップアップを表示する

1.2 WHEN ユーザーが「確認しました」ボタンを押してポップアップを閉じる THEN システムは警告確認状態を保存しない

1.3 WHEN ユーザーが固定資産税路線価フィールドを再度クリックまたは入力する THEN システムは同じ警告ポップアップを再度表示する

1.4 WHEN ユーザーが何度「確認しました」ボタンを押しても THEN システムは固定資産税路線価フィールドへの入力のたびに警告ポップアップを繰り返し表示する

### Expected Behavior (Correct)

2.1 WHEN 固定資産税路線価フィールドに値を入力し、土地面積が99㎡以下または建物面積より小さい場合 THEN システムは初回のみ警告ポップアップを表示する

2.2 WHEN ユーザーが「確認しました」ボタンを押してポップアップを閉じる THEN システムは警告確認状態を保存する

2.3 WHEN ユーザーが固定資産税路線価フィールドを再度クリックまたは入力する THEN システムは警告ポップアップを表示しない

2.4 WHEN ユーザーが「確認しました」ボタンを押した後 THEN システムは固定資産税路線価フィールドへの入力を妨げない

### Unchanged Behavior (Regression Prevention)

3.1 WHEN 土地面積が100㎡以上かつ建物面積以上の場合 THEN システムは警告ポップアップを表示しない

3.2 WHEN 固定資産税路線価フィールドに値を入力する THEN システムは引き続き自動計算機能を実行する

3.3 WHEN ページをリロードする THEN システムは警告確認状態をリセットし、次回の入力時に再度警告を表示する

3.4 WHEN 別の売主の通話モードページを開く THEN システムは新しい売主に対して警告確認状態をリセットする
