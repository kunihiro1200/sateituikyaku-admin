# バグフィックス要件ドキュメント

## はじめに

物件詳細画面（売主管理システム）のヘッダーにある「買主候補」ボタンに2つの問題が発生しています：

1. **パフォーマンス問題**: ボタンをクリックしてから買主候補一覧が表示されるまで約50秒かかる（約10件の買主候補を表示するのに）
2. **表示サイズ問題**: 新しいタブで開かれるウィンドウのサイズが小さく、使いにくい

これらの問題により、ユーザーは買主候補を確認する際に長時間待たされ、表示されたウィンドウも見づらい状態になっています。

## バグ分析

### 現在の動作（不具合）

#### 1.1 パフォーマンス問題
1.1 WHEN 物件詳細画面のヘッダーにある「買主候補」ボタンをクリックした時 THEN システムは約50秒後に買主候補一覧を表示する

1.2 WHEN 買主候補が約10件程度の場合でも THEN システムは毎回約50秒の遅延が発生する

#### 1.2 表示サイズ問題
1.3 WHEN 「買主候補」ボタンをクリックして新しいタブが開かれた時 THEN システムは小さいサイズのウィンドウで買主候補一覧を表示する

### 期待される動作（正しい動作）

#### 2.1 パフォーマンス改善
2.1 WHEN 物件詳細画面のヘッダーにある「買主候補」ボタンをクリックした時 THEN システムは3秒以内に買主候補一覧を表示する

2.2 WHEN 買主候補が約10件程度の場合 THEN システムは高速に（3秒以内に）買主候補一覧を表示する

#### 2.2 表示サイズ改善
2.3 WHEN 「買主候補」ボタンをクリックして新しいタブが開かれた時 THEN システムは適切なサイズのウィンドウで買主候補一覧を表示する

### 変更されない動作（リグレッション防止）

#### 3.1 機能の保持
3.1 WHEN 「買主候補」ボタンをクリックした時 THEN システムは引き続き物件所在地周辺で問合せがあった方や、そのエリアを希望している買主の一覧を表示する

3.2 WHEN 買主候補一覧が表示された時 THEN システムは引き続きテーブル形式で買主情報を表示する

3.3 WHEN 「買主候補」ボタンをクリックした時 THEN システムは引き続き新しいタブで買主候補一覧を開く

#### 3.2 他の機能への影響なし
3.4 WHEN 物件詳細画面の他のボタンやフィールドを操作した時 THEN システムは引き続き正常に動作する

3.5 WHEN 買主候補一覧以外のページを表示した時 THEN システムは引き続き正常に動作する
