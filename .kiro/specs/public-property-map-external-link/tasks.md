# 実装計画：公開物件サイト地図マーカー外部リンク機能

## 概要

PropertyMapViewコンポーネントの「詳細を見る」ボタンの動作を変更し、新しいタブで物件詳細ページを開くようにします。

## タスク

- [x] 1. PropertyMapViewコンポーネントの修正
  - `handlePropertyClick`関数を`window.open()`を使用するように変更
  - `navigate`と`useNavigate`のインポートを削除
  - ボタンに`aria-label`属性を追加してアクセシビリティを向上
  - _要件: 1.1, 1.2, 1.3, 2.1, 3.1, 3.2, NFR-2_

- [ ]* 1.1 ユニットテストの作成
  - `handlePropertyClick`関数が`window.open`を正しい引数で呼び出すことをテスト
  - モックを使用して`window.open`の呼び出しを検証
  - _要件: 1.1_

- [x] 2. 動作確認
  - ローカル環境で地図を表示し、マーカーをクリック
  - 「詳細を見る」ボタンをクリックして新しいタブで開くことを確認
  - 元のタブで地図画面が保持されていることを確認
  - 複数の物件で動作を確認
  - _要件: 1.1, 1.2, 1.3, 2.1, 2.2_

- [x] 3. チェックポイント
  - すべてのテストが通過することを確認
  - 動作確認で問題がないことを確認
  - 質問があればユーザーに確認

## 注意事項

- タスク1は必須の実装タスクです
- タスク1.1はオプションのテストタスクです（`*`マーク付き）
- タスク2は手動での動作確認です
- 既存の地図表示機能に影響を与えないよう、最小限の変更に留めます
