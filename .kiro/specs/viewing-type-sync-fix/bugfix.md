# Bugfix Requirements Document

## Introduction

内覧リスト（物件番号: 7344）において、スプレッドシートに入力されている「内覧形態」の値がDBに反映されていないバグを修正する。スプレッドシート→DB同期処理において「内覧形態」フィールドが書き込まれないため、DB上の該当フィールドが常に空のままとなっている。

## Bug Analysis

### Current Behavior (Defect)

1.1 WHEN スプレッドシートの「内覧形態」列に値が入力されている場合 THEN システムはスプレッドシート→DB同期処理実行時にその値をDBの対応フィールドに書き込まない

1.2 WHEN スプレッドシート→DB同期処理が実行された場合 THEN システムは「内覧形態」フィールドの値をDBに反映せず、DBの該当フィールドが空のままになる

### Expected Behavior (Correct)

2.1 WHEN スプレッドシートの「内覧形態」列に値が入力されている場合 THEN システムはスプレッドシート→DB同期処理実行時にその値をDBの対応フィールドに正しく書き込む

2.2 WHEN スプレッドシート→DB同期処理が実行された場合 THEN システムは「内覧形態」フィールドの値をDBに保存する

### Unchanged Behavior (Regression Prevention)

3.1 WHEN 「内覧形態」以外のフィールドのスプレッドシート→DB同期処理が実行された場合 THEN システムはそれらのフィールドを引き続き正しくDBに反映する

3.2 WHEN スプレッドシートの「内覧形態」列が空欄の場合 THEN システムはDBの該当フィールドも空のまま維持する

3.3 WHEN 内覧リスト以外の同期処理（売主リスト、買主リストなど）が実行された場合 THEN システムはそれらの同期処理に影響を与えない
