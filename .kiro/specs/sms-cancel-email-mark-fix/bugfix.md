# Bugfix Requirements Document

## Introduction

通話モードページのSMS送信機能において、「キャンセル案内」テンプレートを選択した際に、「メール送信確認」セクションの「査定理由別３後Eメ担当」に不要な赤色マークが表示される問題を修正します。

この問題により、ユーザーは誤って「査定理由別３後Eメ担当」にもマークが付いていると認識してしまい、実際には関係のない担当者に通知が行くと誤解する可能性があります。

## Bug Analysis

### Current Behavior (Defect)

1.1 WHEN SMS送信セクションで「キャンセル案内」テンプレートを選択 THEN 「メール送信確認」セクションの「査定理由別３後Eメ担当」のスタッフイニシャルボタンに赤色のマークが表示される

1.2 WHEN SMS送信セクションで「キャンセル案内」テンプレートを選択 THEN 「メール送信確認」セクションの「キャンセル案内担当」のスタッフイニシャルボタンに赤色のマークが表示される（これは正しい）

### Expected Behavior (Correct)

2.1 WHEN SMS送信セクションで「キャンセル案内」テンプレートを選択 THEN 「メール送信確認」セクションの「査定理由別３後Eメ担当」にはマークが表示されない

2.2 WHEN SMS送信セクションで「キャンセル案内」テンプレートを選択 THEN 「メール送信確認」セクションの「キャンセル案内担当」のみに赤色のマークが表示される

### Unchanged Behavior (Regression Prevention)

3.1 WHEN Email送信で「（査定理由別）～」のテンプレートを送信 THEN 「査定理由別３後Eメ担当」に青色のマークが正しく表示される

3.2 WHEN SMS送信で「不通時Sメール」などの他のテンプレートを選択 THEN 「査定理由別３後Eメ担当」に不要なマークは表示されない

3.3 WHEN Email送信で「キャンセル案内」テンプレートを送信 THEN 「キャンセル案内担当」に青色のマークが正しく表示される

3.4 WHEN SMS送信で「キャンセル案内」以外のテンプレートを選択 THEN 対応する担当者のみに赤色のマークが表示される
