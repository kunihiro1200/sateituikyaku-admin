# バグ修正要件ドキュメント

## Introduction

買主リスト7260において、スプレッドシートの「業者向けアンケート」フィールドが「確認済み」になっているにもかかわらず、サイドバーに「業者問合せあり」として表示される問題を修正します。

この問題により、実際には対応済みの買主が未対応として表示され、業務効率が低下しています。

## Bug Analysis

### Current Behavior (Defect)

1.1 WHEN 買主の「業者向けアンケート」フィールドが「確認済み」である THEN サイドバーに「業者問合せあり」として誤って表示される

1.2 WHEN 買主番号7260のスプレッドシートデータを確認する THEN 「業者向けアンケート」列は"確認済み"だが、サイドバーには「業者問合せあり」カテゴリに含まれている

### Expected Behavior (Correct)

2.1 WHEN 買主の「業者向けアンケート」フィールドが「確認済み」である THEN サイドバーの「業者問合せあり」カテゴリに表示されない

2.2 WHEN 買主の「業者向けアンケート」フィールドが「未」である THEN サイドバーの「業者問合せあり」カテゴリに表示される

### Unchanged Behavior (Regression Prevention)

3.1 WHEN 買主の「業者向けアンケート」フィールドが「未」である THEN システムは引き続き「業者問合せあり」カテゴリに表示する

3.2 WHEN 買主の「業者向けアンケート」フィールドが空である THEN システムは引き続き現在の表示ロジックを維持する

3.3 WHEN 他のサイドバーカテゴリ（「内覧日前日」「当日TEL」など）の条件を満たす買主がいる THEN システムは引き続き正しく表示する
