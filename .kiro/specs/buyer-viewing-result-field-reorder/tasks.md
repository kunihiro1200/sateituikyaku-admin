# 実装計画: 内覧後売主連絡フィールドの配置変更

## 概要

`BuyerViewingResultPage.tsx` において、「内覧後売主連絡」フィールドをヒアリング項目セクションから内覧情報行（内覧未確定フィールドの右隣）へ移動する。変更対象はフロントエンドの1ファイルのみ。

## タスク

- [x] 1. 内覧情報行への内覧後売主連絡フィールドの移動
  - [x] 1.1 内覧情報行（`display: flex` の Box）内の内覧未確定フィールド直後に、内覧後売主連絡フィールドのJSXブロックを追加する
    - 一般媒介条件（`viewing_mobile` または `viewing_type_general` に「一般」を含む）を満たす場合のみ表示
    - ラベル「内覧後売主連絡」を表示し、必須条件を満たす場合は「*」を付与
    - 「済」「未」「不要」の3ボタンを横並びで表示
    - 選択済みボタンは `contained` スタイルで強調表示
    - 選択済みボタンの再クリックで `post_viewing_seller_contact` をクリア
    - 注意書き「*一般媒介は内覧後に、全ての売り主に結果報告をしてください」をフィールド下部に表示
    - _Requirements: 1.1, 1.2, 1.3, 1.5, 2.1, 2.2, 2.3, 2.4, 2.5, 2.6_

  - [x] 1.2 ヒアリング項目セクション内の内覧後売主連絡フィールドのJSXブロック（`{/* 内覧後売主連絡 */}` コメントから始まる `{(() => { ... })()}` ブロック全体）を削除する
    - _Requirements: 1.4_
