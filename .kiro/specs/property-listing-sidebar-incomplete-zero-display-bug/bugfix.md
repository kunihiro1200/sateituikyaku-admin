# バグ修正要件ドキュメント

## Introduction

物件リストページ（PropertyListingsPage）のサイドバー（PropertySidebarStatus）において、「未完了」カテゴリの件数が0件であるにもかかわらず、カテゴリ自体がサイドバーに表示されている問題を修正します。

ユーザーの期待動作として、件数が0件のカテゴリはサイドバーに表示されないべきです。これにより、サイドバーが不要な情報で混雑することを防ぎ、ユーザーが実際に対応が必要なカテゴリに集中できるようになります。

## Bug Analysis

### Current Behavior (Defect)

1.1 WHEN 「未完了」カテゴリの件数が0件の場合 THEN サイドバーに「未完了」カテゴリが表示される（件数バッジは非表示）

1.2 WHEN 任意のカテゴリの件数が0件の場合 THEN そのカテゴリがサイドバーに表示される（件数バッジは非表示）

### Expected Behavior (Correct)

2.1 WHEN 「未完了」カテゴリの件数が0件の場合 THEN サイドバーに「未完了」カテゴリは表示されない

2.2 WHEN 任意のカテゴリの件数が0件の場合 THEN そのカテゴリはサイドバーに表示されない

### Unchanged Behavior (Regression Prevention)

3.1 WHEN 「未完了」カテゴリの件数が1件以上の場合 THEN サイドバーに「未完了」カテゴリが表示され、件数バッジも表示される

3.2 WHEN 任意のカテゴリの件数が1件以上の場合 THEN そのカテゴリがサイドバーに表示され、件数バッジも表示される

3.3 WHEN 「すべて」カテゴリが選択されている場合 THEN 全ての物件が表示される

3.4 WHEN 件数が1件以上のカテゴリを選択した場合 THEN そのカテゴリに該当する物件のみが表示される
