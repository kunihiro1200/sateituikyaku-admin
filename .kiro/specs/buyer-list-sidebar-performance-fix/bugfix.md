# Bugfix Requirements Document

## Introduction

買主リストページを開いた際、サイドバーのカテゴリー（ステータス別件数など）が表示されるまでに約20秒かかるパフォーマンス問題。以前は正常に動作していたことから、何らかの変更によって性能が劣化したと考えられる。ユーザーの作業効率に直接影響する重大な問題。

## Bug Analysis

### Current Behavior (Defect)

1.1 WHEN ユーザーが買主リストページを開く THEN サイドバーのカテゴリー（ステータス別件数）が表示されるまでに約20秒かかる

1.2 WHEN サイドバーのカテゴリーデータを取得・計算する THEN システムが異常に長い処理時間を要し、ユーザーを長時間待機させる

### Expected Behavior (Correct)

2.1 WHEN ユーザーが買主リストページを開く THEN サイドバーのカテゴリーは数秒以内（5秒未満）に表示される

2.2 WHEN サイドバーのカテゴリーデータを取得・計算する THEN システムは適切な速度で処理を完了し、ユーザーを不必要に待機させない

### Unchanged Behavior (Regression Prevention)

3.1 WHEN サイドバーのカテゴリーが表示される THEN システムは各カテゴリーの件数を正確に表示し続ける

3.2 WHEN 買主データが更新される THEN システムはサイドバーのカテゴリー件数に変更を反映し続ける

3.3 WHEN ユーザーがサイドバーのカテゴリーをクリックしてフィルタリングする THEN システムは正しいフィルタリング結果を表示し続ける

3.4 WHEN 買主リストページの一覧表が表示される THEN システムは既存の一覧表示機能を正常に動作させ続ける
