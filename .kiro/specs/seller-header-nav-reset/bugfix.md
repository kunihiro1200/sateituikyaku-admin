# Bugfix Requirements Document

## Introduction

売主管理システムのヘッダーにある「売主リスト」ボタンを押しても、売主一覧のトップページ（全件表示状態）に戻らないバグ。
サイドバーでカテゴリーフィルターが適用された状態でヘッダーの「売主リスト」ボタンを押しても、フィルターがリセットされず全件表示に戻れない。
本来、どのページからでもヘッダーの「売主リスト」ボタンを押したら、フィルターなしの全件表示状態の売主一覧トップに遷移すべきである。

## Bug Analysis

### Current Behavior (Defect)

1.1 WHEN サイドバーのカテゴリー（例：「当日TEL分」）をクリックしてフィルタリングされた売主一覧ページを表示している状態で、ヘッダーの「売主リスト」ボタンを押した THEN the system はフィルターをリセットせず、現在のフィルタリング状態のまま表示し続ける

1.2 WHEN フィルタリングされた売主一覧ページでヘッダーの「売主リスト」ボタンを押した THEN the system は売主一覧のトップ（全件表示）に遷移しない

1.3 WHEN 売主詳細ページや通話モードページなどの売主一覧以外のページでヘッダーの「売主リスト」ボタンを押した THEN the system は売主一覧に遷移するが、以前のフィルター状態が残っている場合がある

### Expected Behavior (Correct)

2.1 WHEN サイドバーのカテゴリーでフィルタリングされた売主一覧ページを表示している状態で、ヘッダーの「売主リスト」ボタンを押した THEN the system SHALL フィルターをリセットして全件表示状態の売主一覧トップに遷移する

2.2 WHEN フィルタリングされた売主一覧ページでヘッダーの「売主リスト」ボタンを押した THEN the system SHALL 売主一覧のトップ（全件表示）に遷移する

2.3 WHEN 売主詳細ページや通話モードページなどの売主一覧以外のページでヘッダーの「売主リスト」ボタンを押した THEN the system SHALL フィルターなしの全件表示状態の売主一覧トップに遷移する

### Unchanged Behavior (Regression Prevention)

3.1 WHEN サイドバーのカテゴリー（例：「当日TEL分」）をクリックした THEN the system SHALL CONTINUE TO そのカテゴリーでフィルタリングされた売主一覧を表示する

3.2 WHEN 「全件表示」ボタンを押した THEN the system SHALL CONTINUE TO フィルターをリセットして全件表示状態の売主一覧を表示する

3.3 WHEN 売主一覧ページで検索やフィルターを操作した THEN the system SHALL CONTINUE TO 指定した条件で売主一覧を絞り込んで表示する

3.4 WHEN ヘッダーの「売主リスト」ボタン以外のナビゲーション（買主リスト、物件リストなど）を押した THEN the system SHALL CONTINUE TO それぞれの対応するページに遷移する
