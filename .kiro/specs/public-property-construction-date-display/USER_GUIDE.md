# ユーザーガイド - 新築年月表示機能

## 機能概要

公開物件サイトにおいて、戸建てとマンションの物件に対して新築年月が表示されるようになりました。この機能により、物件の築年数を容易に把握でき、より適切な物件選択が可能になります。

## 表示場所

### 1. 物件一覧ページ

物件カードの物件特徴セクション（土地面積、建物面積などが表示されている場所）の最初に新築年月が表示されます。

**表示例:**
```
📅 2020年03月
🏠 土地: 150㎡
🏢 建物: 100㎡
```

### 2. 物件詳細ページ

物件詳細セクションの最初に新築年月が表示されます。

**表示例:**
```
新築年月
2020年03月

土地面積
150㎡

建物面積
100㎡
```

## 表示される物件タイプ

新築年月は以下の物件タイプでのみ表示されます:

- ✅ **一戸建て（戸建て）**
- ✅ **マンション**

以下の物件タイプでは表示されません:

- ❌ **土地**: 建物がないため新築年月は関係ありません
- ❌ **その他**: 収益物件など、建物の築年数が主要な情報でない物件

## 日付フォーマット

新築年月は常に「YYYY年MM月」形式で表示されます。

**表示例:**
- `2020年03月`
- `1999年08月`
- `2024年05月`

月は常に2桁で表示されます（例: 3月 → 03月）。

## データがない場合

新築年月のデータがない物件では、このフィールドは表示されません。レイアウトは自動的に調整され、他の情報が適切に表示されます。

## よくある質問（FAQ）

### Q1: なぜ土地物件には新築年月が表示されないのですか？

**A:** 土地には建物がないため、新築年月という概念がありません。土地物件では新築年月フィールドを非表示にしています。

### Q2: 新築年月と築年数の違いは何ですか？

**A:** 
- **新築年月**: 建物が建築された年月（例: 2020年03月）
- **築年数**: 現在から見た建物の経過年数（例: 築5年）

両方の情報が表示される場合もあります。

### Q3: 新築年月が表示されない戸建て・マンション物件があるのはなぜですか？

**A:** 以下の理由が考えられます:
1. データベースに新築年月の情報が登録されていない
2. データの形式が認識できない

このような場合、新築年月フィールドは非表示になりますが、他の情報は正常に表示されます。

### Q4: 新築年月で物件を検索・フィルタリングできますか？

**A:** 現在のバージョンでは、新築年月による検索・フィルタリング機能は実装されていません。今後のアップデートで追加される可能性があります。

### Q5: モバイルでも新築年月は表示されますか？

**A:** はい、モバイル、タブレット、デスクトップのすべてのデバイスで新築年月が表示されます。レスポンシブデザインにより、画面サイズに応じて適切にレイアウトされます。

## トラブルシューティング

### 問題: 新築年月が表示されない

**確認事項:**
1. 物件タイプが戸建てまたはマンションであることを確認
2. 他の物件情報（土地面積、建物面積など）が表示されているか確認
3. ブラウザのキャッシュをクリアして再読み込み

### 問題: 新築年月の形式がおかしい

**対処法:**
1. ブラウザのキャッシュをクリアして再読み込み
2. 問題が解決しない場合は、サポートに連絡してください
