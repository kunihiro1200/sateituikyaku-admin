# ページネーション問題デバッグチェックリスト（絶対に守るべきルール）

## ⚠️ 重要：今回の問題で時間がかかった原因

**問題**: 「訪問日前日」カテゴリのサイドバーカウント（2件）と一覧表示件数（1件→全件7109件）が不一致

**解決に時間がかかった原因**:

### 原因1: 2つの独立したメソッドを使用していることに気づかなかった

- **サイドバーカウント**: `getSidebarCountsFallback()` メソッド
- **一覧フィルタリング**: `listSellers()` メソッド

**問題**: 最初は「同じロジックを使っているはず」と思い込んでいた

**対策**: 
- ✅ **最初に両方のメソッドを確認する**
- ✅ **「サイドバーと一覧は別々のメソッドを使用している」という前提で調査する**

---

### 原因2: ページネーション問題を見逃した

**問題**: Supabaseのデフォルト制限（1000件）により、AA13729（1001件目以降）が取得できていなかった

**なぜ見逃したか**:
- データベースには2件とも存在していた
- SQLクエリの条件は正しかった
- **「1000件制限」という前提知識が欠けていた**

**対策**:
- ✅ **Supabaseのデフォルト制限（1000件）を常に意識する**
- ✅ **全件取得が必要な場合は、必ずページネーション処理を実装する**
- ✅ **「データベースに存在する」≠「クエリで取得できる」という認識を持つ**

---

### 原因3: 存在しないメソッド（fetchAllRows）を呼び出していた

**問題**: `listSellers()` で `this.fetchAllRows()` を呼び出していたが、このメソッドは定義されていなかった

**なぜ気づかなかったか**:
- TypeScriptのコンパイルエラーが出なかった（型チェックが甘かった）
- ローカルテストを実行しなかった
- **「コードが正しく見える」≠「実際に動作する」という認識が欠けていた**

**対策**:
- ✅ **修正後は必ずローカルでテストする**
- ✅ **存在しないメソッドを呼び出していないか確認する**
- ✅ **TypeScriptの型チェックを厳格にする**

---

### 原因4: 段階的なデバッグができていなかった

**問題**: 
1. サイドバーカウントを修正（ページネーション追加）
2. デプロイ
3. 一覧が直っていないことに気づく
4. また調査...

**なぜ時間がかかったか**:
- **サイドバーと一覧を別々に修正した**
- **最初に両方のメソッドを比較しなかった**

**対策**:
- ✅ **最初に全体像を把握する**（サイドバーと一覧の両方のコードを確認）
- ✅ **同じロジックを使用すべき箇所は、最初から両方を修正する**
- ✅ **「片方だけ修正すれば良い」という思い込みを捨てる**

### ステップ2: ページネーション問題を確認する

**質問**:
- [ ] 全件取得が必要な処理か？
- [ ] Supabaseのデフォルト制限（1000件）を超える可能性があるか？
- [ ] ページネーション処理が実装されているか？

**確認方法**:
```typescript
// ❌ 悪い例（1000件制限）
const { data } = await supabase.from('sellers').select('*');

// ✅ 良い例（全件取得）
let allRows: any[] = [];
let page = 0;
const pageSize = 1000;

while (true) {
  const { data, error } = await supabase
    .from('sellers')
    .select('*')
    .range(page * pageSize, (page + 1) * pageSize - 1);
  
  if (error || !data || data.length === 0) break;
  
  allRows = allRows.concat(data);
  
  if (data.length < pageSize) break;
  page++;
}
```

### ステップ4: ローカルでテストする

**テスト手順**:
1. テストスクリプトを作成
2. 環境変数を読み込む
3. 実際のデータで動作確認
4. 期待される結果と一致するか確認

**例**:
```typescript
// backend/test-visit-day-before.ts
import { SellerService } from './src/services/SellerService.supabase';

async function test() {
  const sellerService = new SellerService();
  
  // サイドバーカウント
  const sidebarCounts = await sellerService.getSidebarCountsFallback();
  console.log('Sidebar count:', sidebarCounts.visitDayBefore);
  
  // 一覧フィルタリング
  const listResult = await sellerService.listSellers({
    statusCategory: 'visitDayBefore',
    page: 1,
    limit: 100,
  });
  console.log('List count:', listResult.total);
  
  // 一致確認
  if (sidebarCounts.visitDayBefore === listResult.total) {
    console.log('✅ SUCCESS: Counts match!');
  } else {
    console.log('❌ FAILED: Counts mismatch!');
  }
}

test().catch(console.error);
```
