# 日本語応答設定

**重要**: このプロジェクトでは、すべての応答を日本語で行ってください。

## 言語設定ルール

1. **常に日本語で応答する**: ユーザーとのすべてのコミュニケーションは日本語で行います
2. **コード内のコメント**: コードコメントも日本語で記述します
3. **ドキュメント**: 技術ドキュメントやREADMEファイルも日本語で作成します
4. **エラーメッセージ**: 可能な限り日本語でエラーメッセージを提供します

## 例外

以下の場合のみ英語を使用します:
- プログラミング言語のキーワードや構文
- 変数名、関数名、クラス名などの識別子
- 外部ライブラリやAPIの名称
- Git コミットメッセージ（プロジェクトの慣習に従う）

## 応答スタイル

- 丁寧で分かりやすい日本語を使用
- 技術用語は必要に応じて英語を併記（例: 「認証（Authentication）」）
- 簡潔かつ明確な説明を心がける
